Но у класса BufferedReader есть метод, который может существенно облегчить работу с текстовыми файлами:
метод для чтения строки из файла readLine(). Данный метод возвращает объект типа String, содержащий очередную прочитанную строку файла.
Если  достигнут конец файла, возвращается значение null. Таким образом – блок чтения файла из предыдущего примера может принять вид
